# Progress 2 — Workflow Telegram → n8n → OpenAI

Progress ini menjelaskan pembuatan workflow Telegram Bot dengan n8n dan OpenAI.

## 1. Membuat Telegram Bot
1. Buka Telegram.
2. Cari BotFather.
3. Jalankan perintah `/newbot`.
4. Dapatkan BOT_TOKEN.
